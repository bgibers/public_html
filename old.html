P4: GPA BusRoutes

Download Download Schemas

My application is an aid to help students know what bus they are too ride back to their living location. It also provides a map view showing the student where the bus stop is located. Simply click the new user button to create an account and register. All info is then stored using core data while the living location is stored to the external database. Upon logging in successfully, the application will tell you what bus you are to take home. Clicking on one of the items in the table will pull up a map view showing you where that bus stop is on campus. All table info including the stop and its corridinates are pulled down from the external database. Whats new: pickerview, mapkit, custom segue transistions, as well as a link from the tableview to the map. Acknowledgments go out to Sydney Pauls P4 as well as Rob Percivals tutorials.

P2: GPA Calculator

Download

My name is Brendan Giberson and this is my app! The main page shows you your grades that you have already inputted. You can calculate your GPA by clicking the button. By clicking the + button you can add more grades to the list of grades. An alert is shown if no grades are entered and the user tries to calculate a GPA. The users current location is shown under the location tab. Whats new? Swipe gesture on table cell to delete a grade, the ability for grades to reload upon opening, case insensitive text field, tab view, camera button to save a screenshot to your camera roll, and shake gesture to delete all grades.
